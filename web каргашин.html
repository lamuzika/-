Контрольные вопросы.

1)Теги "tаble", "tr" и "td" обязательны при создании HTML-таблицы. "table" определяет саму таблицу, "tr" – строки, а "td" – ячейки данных внутри строк.
Если нужен заголовок, можно использовать также тег "th" вместо "td" в первой строке таблицы.

2)Границы таблицы в HTML задаются с помощью свойства "border". Это свойство может быть применено к тегу "table" для определения толщины границы всей таблицы.
Например, "border="1"" устанавливает границу толщиной в 1 пиксель.

3)Тег "td" используется для создания ячеек данных внутри HTML-таблицы, предназначенных для обычного содержимого. Тег "th", напротив, используется для создания заголовочных ячеек, обозначая важность содержимого и стилизуя его более выделенно (жирным шрифтом и выравниванием по центру). Заголовочные ячейки обычно используются для описания данных в столбцах или строках таблицы.

4)Для объединения ячеек по горизонтали используется атрибут `colspan`, а для объединения по вертикали – `rowspan`. Атрибут `colspan` указывает, сколько ячеек таблицы должно быть объединено в текущей строке, а `rowspan` определяет, сколько ячеек таблицы должно быть объединено в текущем столбце. Эти атрибуты применяются к тегам "td" или "th".

5)Цвет фона ячеек таблицы в HTML можно задать с использованием атрибута `style` и свойства `background-color`. Например, "td style='background-color: lightblue'"" установит светло-голубой цвет фона для ячейки. Можно также использовать CSS классы или внешние таблицы стилей для более сложных и структурированных стилей.
